# 远程对比框渲染方案

## 背景

H3 BattleValueInfo 插件在战场顶部显示一个远程/法术对比面板（298×93 像素），需要同时显示背景图和白色数字。

## 问题

最初使用 DirectDraw 的 `Lock()` + 直接写入像素 + `Unlock()` 方式，在原版 SoD 可行，但在 HD 版（使用 OpenGL 渲染层）中：
- backbuffer 的像素格式是 BGRA（A=0，即完全透明）
- 直接写入像素后颜色完全不对
- `Blt()` 到 backbuffer 会触发 HD_VGL2 崩溃

## 最终方案

**完全不使用 DirectDraw**，所有绘制都走游戏原生 `screen_pcx16`（16bpp RGB565 surface），最后用 `H3Redraw` 强制 GPU 刷新对比框区域。

### 核心流程

```
PCX 文件 (8bpp indexed color)
    ↓ RLE 解码 + palette 查表 (RGB565)
_Pcx16_ bg_ 对象
    ↓ 复制到临时合成图 bg_composite_
_Pcx16_ bg_composite_ 对象 (298×93)
    ↓ 文字 TextDraw() 叠加到合成图
bg_composite_ (背景 + 文字)
    ↓ DrawPcx16ToBackBuffer() 一次 blt
backbuffer
    ↓ H3Redraw() 强制 GPU 刷新
屏幕显示
```

### 关键设计

1. **双缓冲结构**：`bg_` 缓存解码后的背景，`bg_composite_` 每帧重建用于合成
2. **合成绘制**：背景和文字先在 `bg_composite_` 上合成，再一次性 blt 到 backbuffer
3. **强制刷新**：`o_WndMgr->H3Redraw(x, y, panel_w, panel_h)` 通知 GPU 刷新面板区域

### 文字布局

面板宽 298 像素，中线在 x=149。两列数字分列中线两侧：
- 左列：x=8，宽 141，右边界贴着中线，MIDDLE_RIGHT 对齐
- 右列：x=154，宽 144，左边界贴着中线，MIDDLE_LEFT 对齐
- Y 坐标从 INI 配置读取（Row1Y/Row2Y/Row3Y）

### 代码流程（DrawImpl）

```cpp
// 1. 创建/重建合成图（尺寸匹配面板）
if (!bg_composite_ || bg_composite_->width != panel_w || bg_composite_->height != panel_h) {
    if (bg_composite_) bg_composite_->Destroy();
    bg_composite_ = _Pcx16_::Create(panel_w, panel_h);
}

// 2. 把背景复制到合成图
for (int yy = 0; yy < panel_h; ++yy) {
    _word_* dst_row = (_word_*)(bg_composite_->buffer + yy * bg_composite_->scanlineSize);
    _word_* src_row = (_word_*)(bg_->buffer + yy * bg_->scanlineSize);
    for (int xx = 0; xx < panel_w; ++xx) dst_row[xx] = src_row[xx];
}

// 3. 文字叠加到合成图
font->TextDraw(bg_composite_, tmp, left_col_x, row_y, left_col_w, row_h,
    NH3Dlg::eTextColor::WHITE, NH3Dlg::eTextAlignment::MIDDLE_RIGHT);
font->TextDraw(bg_composite_, tmp, right_col_x, row_y, right_col_w, row_h,
    NH3Dlg::eTextColor::WHITE, NH3Dlg::eTextAlignment::MIDDLE_LEFT);

// 4. 一次 blt 到 backbuffer
DrawPcx16ToBackBuffer(bg_composite_, x, y, FALSE);

// 5. 强制刷新面板区域
o_WndMgr->H3Redraw(x, y, panel_w, panel_h);
```

## 历史

- 最初用 DD Lock/Write，原版正常，HD 版颜色不对
- 尝试 Blt 到 backbuffer，HD 版崩溃
- 最终方案：不用 DD，所有绘制走 screen_pcx16，H3Redraw 刷新
