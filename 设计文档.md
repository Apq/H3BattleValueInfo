# BattleValueInfo 设计文档

> 本文描述当前实现。窗口扩展、背景、按钮和描述区布局已迁移到 MegaDesc。

## 1. 功能范围

BattleValueInfo 只负责：

- 在生物信息窗口中追加战斗价值行；
- 在战斗中右键空白格时显示双方远程兵种力量对比。

不再负责：

- 生物信息窗口加高；
- 背景替换；
- 按钮替换；
- 描述区扩展和窗口位置吸附。

这些功能由 MegaDesc 项目负责。

## 2. 战斗价值显示

战斗价值来源：

```text
*(int*)(*(char**)0x6747B0 + creature_id * 0x74 + 0x3C)
```

其中：

- 全局生物表指针：`0x6747B0`
- `_CreatureInfo_` 结构大小：`0x74`
- `fight_value` 偏移：`+0x3C`

显示值：

```text
总战斗价值 = 单只 fight_value × 当前数量
```

窗口中新增控件：

- label id：`3009`
- value id：`3008`

如果已存在上述控件，则跳过，避免重复插入。

## 3. Hook 列表

| Hook | 地址 | 类型 | 说明 |
|------|------|------|------|
| BuildCombat | 0x5F4503 | LoHook | 战斗生物信息窗口构建后插入战斗价值 |
| BuildAdventure | 0x5F3E75 | LoHook | 冒险生物信息窗口构建后插入战斗价值 |
| BuildTown | 0x5F491E | LoHook | 城镇生物信息窗口构建后插入战斗价值 |
| DlgDefProc | 0x41B120 | HiHook | 兜底补插战斗价值 |
| ShowStatsEntry | 0x468440 | HiHook | 标记右键是否已命中原版战斗格信息 |
| RangedPower | 0x4746B0 | HiHook | 右键空白格时显示远程力量对比 |

## 4. 配置文件

```ini
[General]
Enabled=1

[CreatureInfo]
ShowFightValue=1
ShowRemainingPercent=0

[RangedPower]
ShowRangedPower=1

[Format]
LabelFightValue=Fight Value
LabelOurSide=Ours
LabelEnemySide=Enemy
```

## 5. 项目依赖

`BattleValueInfo.vcxproj` 通过 `ProjectReference` 依赖：

```text
..\H3MegaDesc\MegaDesc.vcxproj
```

构建顺序：先 MegaDesc，后 BattleValueInfo。
